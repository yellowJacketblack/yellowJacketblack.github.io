Game Search Trends Analytics Dashboard

Interactive R Shiny dashboard analyzing gaming search trends

Project Overview

This interactive dashboard analyzes Google Trends data for popular gaming search terms, providing real-time insights into search interest patterns, regional distribution, and emerging query trends. Built as a production-grade analytics tool, it helps content creators and gaming industry professionals identify trending topics and optimize their content strategy.

The dashboard covers the search terms: hunt, new world, ark, smash, and rust, analyzing data from April 20, 2026 to July 20, 2026 across the United States.

Key Features

  • Search Term Selection: Interactive dropdown to analyze individual game terms
  • Key Metrics: Total data points, average search interest, and peak search interest
  • Search Interest Over Time: Daily search interest visualization with relative scoring (0-100)
  • All Terms Comparison: Side-by-side trend comparison across all 5 game terms
  • Regional Interest Distribution: State-by-state search interest breakdown
  • Related Queries Analysis: Top queries and rising queries identification
  • Most Frequent Related Queries: Word cloud visualization for selected terms
  • Basic Statistics: Statistical summary of search interest data
  • Regional Interest Comparison: Multi-term regional analysis
  • Growth Strategy Tips: Actionable insights based on rising query trends
